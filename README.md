# Custom Theme Helpers - Tag Styles

Assign tags to a set of pre-defined color styles.

## Settings

- `yellow`, `orange`, `red`, `green`, `lightblue`, `blue`, `black`: Assign tags to a preset fill color.
- `outline`: Assign tags to outline-only styling.
- `color_target`: Choose whether preset/custom colors apply to the full tag (`tag`) or only the flair square (`flair`).
- `custom_tag_colors`: Add custom mappings in the format `tag-name:#RRGGBB`, separated by `|`.

Example:

`urgent:#c62828|needs-review:#1565c0|blocked:#6a1b9a`

For more information, please see: https://meta.discourse.org/t/colored-tag-styles/214871
